# house_price_linear_regression
# House Price Prediction using Linear Regression 
## Algorithm 
Linear Regression 
This is a beginner-friendly machine learning project for predicting house prices. 
## Features - area - bedrooms - bathrooms - stories 
13 | P a g e 
- parking - optional categorical features 
## Target - price 
## Project Files - `train_model.py` → model training - `app.py` → Streamlit app - `data/housing.csv` → dataset - `models/linear_model.pkl` → saved model - `models/feature_columns.pkl` → training columns 
## How to Run 
### Install packages 
pip install -r requirements.txt 
Train model 
python train_model.py 
Run app 
streamlit run app.py ---
